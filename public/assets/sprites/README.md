# Sprite Assets

## Naming Convention
```
sprite_<stage>_<emotion>_<frame>.png
```

## Parameters
- **stage**: egg, baby, teen, adult
- **emotion**: idle, happy, sad, hungry, sleepy, excited
- **frame**: 01, 02, 03, ...

## Size
All sprites must be **72×72px** PNG with transparent background.

## Example
```
sprites/egg/sprite_egg_idle_01.png
sprites/egg/sprite_egg_idle_02.png
sprites/baby/sprite_baby_happy_01.png
sprites/adult/sprite_adult_excited_01.png
```
