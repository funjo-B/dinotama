1. 시작하기

DinoTama를 설치하면 화면 우측 하단에 작은 공룡 창(320x280)이 나타납니다. 이 창은 항상 최상위에 표시되며, 시스템 트레이에 상주합니다.

🔐 Google 로그인 우클릭 메뉴
공룡 창에서 우클릭 → "Google 로그인" 클릭. 브라우저가 열리고 Google 계정으로 로그인하면 앱으로 자동 복귀합니다. 로그인하면 클라우드 동기화, Google Calendar 연동, 광고 보상이 활성화됩니다.
로그인하면 우측 상단에 이름과 초록색 점이 표시됩니다.

2. 공룡 기본 조작

👆 공룡 클릭 좌클릭
공룡을 클릭하면 랜덤한 idle 애니메이션 변형이 재생됩니다. 가만히 둬도 8~20초 간격으로 자동으로 움직입니다.

감정 액션 버튼

화면 하단에 감정 액션 버튼이 표시됩니다. 공룡의 레어도에 따라 사용 가능한 버튼 수가 달라집니다.

버튼액션감정지속시간필요 레어도
🍖먹이기happy4초Common+
🎮놀아주기excited5초Rare+
🤗쓰다듬기happy3초Epic+
😴재우기sleepy5초Legend+
💃춤추기excited6초Hidden
✏️ 이름 변경 우클릭 메뉴
우클릭 → "이름 변경" 선택. 공룡 위에 입력창이 나타나면 새 이름을 입력하고 Enter. 최대 20자.
🔄 공룡 선택 우클릭 메뉴
여러 마리를 보유 중이면 우클릭 → "공룡 선택" 서브메뉴에서 활성 공룡을 변경할 수 있습니다.

3. 가챠 (뽑기)

🥚 가챠 패널 열기 좌측 🥚 버튼 또는 우클릭
좌측 사이드바의 🥚 버튼을 클릭하거나, 우클릭 메뉴에서 "가챠"를 선택하면 가챠 패널이 열립니다.

뽑기 종류

뽑기비용설명
1회 뽑기💰 10알 1개 부화 연출
5연 뽑기💰 50알 5개 그리드, 개별 클릭으로 공개
10연 뽑기💰 100알 10개 그리드, 모두열기 가능

뽑기 연출

  • 1회 뽑기: 알이 흔들리며 클릭하면 부화. Legend/Hidden 등급은 황금빛 번쩍 이펙트 후 공개.
  • 다연 뽑기: 알 그리드가 나타나고, 각 알을 탭하면 개별 부화. "모두 열기" 버튼으로 한번에 공개.
  • Legend/Hidden 연출: 황금(Legend) 또는 붉은(Hidden) 빛의 특수 이펙트가 2초간 재생된 후 공개됩니다.
  • 다연에서 Legend+Hidden 동시 등장 시: "모두 열기"를 누르면 먼저 나오는(앞번호) 특수 등급 1마리만 플래시 연출 후, 나머지는 즉시 공개됩니다. 개별 클릭 시에는 각각 플래시가 재생됩니다.

천장제 (Pity System)

운이 나쁘더라도 일정 횟수 내에 높은 등급이 보장됩니다:

등급천장의미
Epic50회50회 뽑기 내에 Epic 이상 1회 보장
Legend100회100회 뽑기 내에 Legend 이상 1회 보장
Hidden300회300회 뽑기 내에 Hidden 1회 보장

가챠 패널 하단에 현재 천장 카운터가 표시됩니다.

4. 컬렉션

📦 컬렉션 패널 좌측 📦 버튼
보유 중인 모든 공룡을 종+스테이지별로 그룹화하여 보여줍니다.

기능

  • 스테이지 필터: 전체 / 🐣 유년기 / 🦕 성장기 / 🦖 성체 필터링
  • 그룹 접기/펼치기: 종별 그룹 헤더를 클릭하면 개별 공룡 목록이 열립니다.
  • 대표 설정: 공룡을 우클릭 → "대표 설정"으로 메인 화면 공룡 변경
  • 이름 변경: 우클릭 → "이름 변경"
💰 되팔기 우클릭 메뉴
공룡을 우클릭 → "되팔기"로 코인을 획득합니다. 마지막 1마리는 판매 불가.

5. 합성 진화

🧬 3마리 합성 컬렉션 패널 ⚡ 버튼
같은 종 + 같은 스테이지의 공룡이 3마리 이상이면 컬렉션 그룹 헤더에 ⚡ 합성 버튼이 나타납니다. 클릭하면 3마리 중 2마리가 소모되고, 1마리가 다음 단계로 진화합니다.

진화 경로

🐣 유년기(baby) → 🦕 성장기(teen) → 🦖 성체(adult)

성체(adult)는 더 이상 합성할 수 없습니다. 성체는 되팔기 가격이 4배!

합성 연출

3마리가 모이는 애니메이션 → 번쩍 이펙트 → 진화된 공룡 등장. 화면을 클릭하면 종료.

6. TODO 리스트

📋 TODO 패널 좌측 📋 버튼
할 일을 추가하고 관리하는 패널입니다. 입력창에 텍스트를 적고 + 버튼 또는 Enter로 추가.

기능

  • 체크박스: 할 일 완료 시 체크. 체크된 항목은 취소선 처리.
  • 매일 초기화: 체크한 항목은 다음 날 자동으로 체크 해제됩니다.
  • 개별 알림 토글: 각 할 일 옆 🔔/🔕 버튼으로 해당 항목의 알림 ON/OFF
  • 전체 알림 토글: 헤더의 🔔 버튼으로 전체 알림 ON/OFF
  • 삭제: 우측 ✕ 버튼으로 항목 영구 삭제
🦕 TODO 알림 자동
설정된 간격(기본 30분)마다 미완료 할 일 중 하나가 랜덤으로 팝업됩니다. "했어!" 버튼으로 완료 처리, "나중에" 버튼으로 다음 알림까지 무시.
알림 간격은 환경설정에서 변경 가능합니다 (끄기 / 15분 / 30분 / 1시간 / 2시간).

7. Google Calendar 연동

Google 로그인 시 자동으로 Google Calendar가 연결됩니다.

📅 일정 보기 TODO 패널 상단
TODO 패널 상단에 당일 캘린더 일정이 표시됩니다. ‹ / › 화살표로 이전/다음 날 일정도 확인 가능. 현재 진행 중인 일정은 초록색으로 "NOW" 표시, 지난 일정은 흐리게 표시됩니다.
🔔 일정 알림 자동 (5분 전)
일정 시작 5분 전에 공룡 창 상단에 알림 팝업이 나타납니다. 일정 제목, 장소, 남은 시간이 표시되며, "OK" 또는 "5분 후" 버튼으로 응답합니다.

8. 광고 보상

📺 광고 보고 10연 뽑기 가챠 패널 하단 버튼
가챠 패널의 뽑기 버튼 아래 "📺 광고 보고 10연뽑기" 버튼을 클릭하면 브라우저가 열립니다. 30초간 광고를 시청한 후 "보상 받기" 버튼을 누르면, 앱으로 돌아와 무료 10연 뽑기가 지급됩니다.

제한 사항

  • 1일 최대 3회까지 이용 가능
  • Google 로그인 필수
  • 보상 토큰은 5분 이내에 사용해야 합니다
코인이 부족할 때 광고 보상으로 무료 뽑기를 받아보세요!

10. 창 관리

드래그 이동

공룡 창의 빈 영역을 마우스로 드래그하면 화면 어디든 이동할 수 있습니다.

📍 위치 관리 우클릭 → 위치
위치 초기화: 화면 우측 하단 기본 위치로 복귀
위치 저장: 현재 위치를 기억
위치 복원: 저장한 위치로 이동

패널 창

TODO, 컬렉션, 가챠, 설정 패널은 공룡 창 왼쪽에 별도 창으로 열립니다. 공룡 창을 드래그하면 패널도 함께 따라옵니다.

시스템 트레이

창을 닫아도 앱은 시스템 트레이에서 계속 실행됩니다. 트레이 아이콘을 우클릭하면 로그인/로그아웃, 보이기/종료 등을 선택할 수 있습니다.

11. 클라우드 동기화

Google 로그인 시 게임 데이터가 자동으로 클라우드에 저장됩니다.

동기화 대상

  • 보유 공룡 목록 및 상태
